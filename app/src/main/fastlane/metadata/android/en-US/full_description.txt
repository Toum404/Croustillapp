Croustill'app is a frontend interface for the public API CROUStillant. This application allows you to access the menu, address, and contact information for nearly 1,000 listed CROUS restaurants.

Croustill'app does not collect any personal data and is NOT an official application of the CROUS or the CROUStillant project.

Why Croustill'app?
- Accessible: A search bar, a filter bar, and the restaurants at your fingertips.
- Convenient: Filter restaurants based on whether they are open, PRM-accessible, and by their region or category.
- Privacy: No sensitive permissions required—only internet access.
- Multilingual: Available in French, English, Spanish, and German (Menus are not translated).
- Transparency: 100% open-source code available for everyone to review, built on an equally open API.
