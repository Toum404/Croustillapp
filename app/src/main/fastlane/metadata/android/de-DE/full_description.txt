Croustill'app ist eine Benutzeroberfläche für die öffentliche API CROUStillant. Diese Anwendung ermöglicht Ihnen den Zugriff auf das Menü, die Adresse und die Kontaktdaten von fast 1.000 registrierten CROUS-Mensen.

Croustill'app sammelt keine personenbezogenen Daten und ist KEINE offizielle Anwendung des CROUS oder des CROUStillant-Projekts.

Warum Croustill'app?
- Zugänglich: Eine Suchleiste, eine Filterleiste und alle Mensen auf einen Blick.
- Praktisch: Filtern Sie geöffnete und barrierefreie Mensen nach Region und Kategorie.
- Datenschutz: Keine sensiblen Berechtigungen erforderlich, nur Internetzugang.
- Mehrsprachig: Französisch, Englisch, Spanisch und Deutsch (Die Menüs selbst sind nicht übersetzt).
- Transparenz: 100 % offener Quellcode, der für jeden einsehbar ist und auf einer ebenfalls offenen API basiert.
